# Portfolio Project – Setup Documentation

## Tools Installed

- **Cursor IDE** 
- **Claude Code** - In Cursor
- **Codex** - In Cursor

## Steps Completed

1. Downloaded and installed Cursor IDE from cursor.com
2. Opened Extensions panel in Cursor (`Ctrl+Shift+X`)
3. Searched and installed **Claude Code** extension – login requires a paid
   subscription, so was unable to log in
4. Searched and installed **Codex** extension – logged in successfully using
   the free plan
5. Logged into my existing GitHub account at github.com
6. Created a new **public repository** named `Portfolio-Project`
7. Pasted the GitHub repository link into the Cursor AI agent – it
   automatically cloned the repository without any manual commands needed
8. Created and edited this README.md file inside Cursor (In the clone repository)
9. Used Cursor's built-in terminal to push the code to GitHub using Git commands:
   - `git add .`
   - `git commit -m "Final Step"`
   - `git push`

## Issues I Ran Into

- **Claude Code login** – Requires a paid subscription. Successfully installed
  the extension but could not log in due to this. 
  
- **Git was new to me** – Had not used Git commands before. Looked up a quick
  tutorial, then ran the commands directly in Cursor's terminal and it worked
  perfectly.

- How to set up a professional AI-powered developer environment from scratch
